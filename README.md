# siteofficial-leyu-resource

本仓库用于归档和发布多个独立的 HTML 页面，作为通用资源存储库使用。不针对任何特定域名或网站，仅提供页面文件本身。

## 项目简介

这是一个静态资源仓库，主要存放各类独立 HTML 页面。页面内容可能涵盖说明文档、工具界面、信息展示等不同用途。仓库本身不提供域名绑定、动态服务或运营行为，仅作为页面文件的版本管理和分发点。

## 目录说明

```
.
├── html/          # 存放 HTML 页面文件
├── assets/        # 页面引用的静态资源（CSS、JS、图片等）
├── docs/          # 相关文档或说明
└── README.md      # 本文件
```

- `html/` 目录下每个子文件夹或独立文件对应一个页面项目。
- `assets/` 中按页面需求组织资源文件，避免跨页面引用污染。
- `docs/` 存放与页面相关的使用说明、版本记录等辅助文档。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开运行。
- 页面之间无依赖关系，可单独部署或引用。
- 归档的页面可能包含示例数据、占位内容或演示功能，请勿用于生产环境。
- 页面文件名或路径不暗示任何外部服务或域名关联。

## 维护说明

- 本仓库由维护者不定期更新，添加或修改页面内容。
- 欢迎提交 Issue 或 Pull Request 来报告问题或建议改进。
- 提交新页面时，请确保文件结构清晰，资源文件归入 `assets/` 对应目录。
- 不建议在本仓库中存放敏感信息、动态后端代码或大型二进制文件。

## 使用方式

您可以直接通过 GitHub 的 Raw 文件链接或克隆仓库到本地后，用浏览器打开 `html/` 下的页面文件。如需部署到自己的服务器，请将整个仓库或所需页面文件复制到您的 Web 目录下。
